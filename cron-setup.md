### рЃЏрЃћрЃЌрЃЮрЃЊрЃў 1: Cron-job.org (рЃБрЃцрЃљрЃАрЃЮ, рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў)

1. **рЃЊрЃљрЃћрЃгрЃўрЃћрЃЌ** https://cron-job.org/
2. **рЃЊрЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃЊрЃўрЃЌ** рЃБрЃцрЃљрЃАрЃЮ рЃљрЃюрЃњрЃљрЃарЃўрЃерЃўрЃЌ

   - **Title**: `Bot Uptime`
   - **URL**: `https://your-bot-name.onrender.com/uptime`

   - **Timezone**: `UTC`
   - **Retry on failure**: `Yes`
   - **Save**

## ­ЪћД рЃЉрЃЮрЃбрЃўрЃА Endpoints

рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЉрЃЮрЃбрЃў рЃњрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњ endpoint-рЃћрЃЉрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА:

- **`/`** - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃАрЃбрЃљрЃбрЃБрЃАрЃў
- **`/ping`** - рЃЏрЃљрЃарЃбрЃўрЃЋрЃў ping
- **`/health`** - рЃАрЃарЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў
- **`/uptime`** - рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў)

## ­ЪЊі рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў

рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў ping-рЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃЌ:
```json
{
  "status": "up",
  "timestamp": "2024-01-15T10:30:00"
}
```

## Рџа№ИЈ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў

1. **Render-рЃўрЃА рЃБрЃцрЃљрЃАрЃЮ рЃЋрЃћрЃарЃАрЃўрЃљ** 15 рЃгрЃБрЃЌрЃў рЃўрЃюрЃљрЃЦрЃбрЃўрЃБрЃарЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃўрЃФрЃўрЃюрЃћрЃЉрЃљ
2. **рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ** рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃюрЃћрЃџрЃў (cold start)
3. **рЃњрЃљрЃарЃћ рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў** рЃБрЃцрЃарЃЮ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃўрЃљ рЃЋрЃўрЃЊрЃарЃћ self-ping
4. **рЃБрЃцрЃљрЃАрЃЮ рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў** рЃќрЃЮрЃњрЃ»рЃћрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃА рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў

## ­Ъј» рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў

1. **рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ Cron-job.org** - рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃАрЃљрЃўрЃЏрЃћрЃЊрЃЮ рЃБрЃцрЃљрЃАрЃЮ рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў
2. **рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ 5 рЃгрЃБрЃЌрЃўрЃљрЃюрЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃў** - рЃБрЃцрЃарЃЮ рЃ«рЃерЃўрЃарЃў рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ
3. **рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃарЃћрЃЉрЃљ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЌ** `/uptime` endpoint-рЃќрЃћ
4. **рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃџрЃЮрЃњрЃћрЃЉрЃў** Render-рЃўрЃА рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃќрЃћ

## ­ЪћЇ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃГрЃарЃљ

### рЃЉрЃЮрЃбрЃў рЃљрЃа рЃърЃљрЃАрЃБрЃ«рЃЮрЃЉрЃА
1. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ Render-рЃўрЃА рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃў
2. рЃюрЃљрЃ«рЃћрЃЌ рЃџрЃЮрЃњрЃћрЃЉрЃў
3. рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃАрЃћрЃарЃЋрЃўрЃАрЃў

### рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА

2. рЃбрЃћрЃАрЃбрЃў рЃњрЃљрЃўрЃљрЃарЃћрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў
3. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў
